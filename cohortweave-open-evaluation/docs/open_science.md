# Open-science scope

The public contribution is the evaluation layer: metric implementations, synthetic fixtures, configuration examples, frozen aggregate results, tests and Responsible AI documentation. The repository is designed for a DOI-bearing tagged release.

The following remain closed: protected data, patient-level outputs, unsafe generations, private model weights, deployment credentials, institutional correspondence and proprietary orchestration. Open science is therefore bounded by privacy, security, dataset licences, competition IPR and future commercial agreements.
